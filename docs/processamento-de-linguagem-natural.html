Processamento de Linguagem Natural

Chapter 2 Processamento de Linguagem Natural

2.1 O que é?

De forma simplificada é fazer com que o computador entenda a comunicação humana, seja pela fala ou escrita.

2.2 Aplicações

  • Respostas a perguntas.
  • Tradução.
  • Correção ortográfica.
  • Análise de sentimentos.
  • Reconhecimento ótico de caracteres (OCR).
  • Reconhecimento de fala.
  • Sintetização de fala.
  • Previsão de digitação.
  • Classificação de textos e documentos.
  • Resumos.

2.3 Níveis de processamento

  • Fonológico: Essa parte corresponde à interpretação dos sons da palavra.

  • Morfológico: Estudo sobre a natureza das palavras e suas composições.

  • Lexical: Processamento responsável por interpretar o significado de cada palavra.

  • Sintático: Estuda a composição da frase.

  • Semântico: Processamento que visa compreender o significado completo da frase.

  • Discurso: Estuda o texto como um todo.

  • Pragmático: Processamento utilizado para buscar significados além das palavras, as entrelinhas.

2.4 Conceitos

  • Corpus: Conjunto de dados (texto não estruturado) em linguagem natural.

  • Annotations:
    • Processo mais importante em PLN.
    • Coloca anotações no texto, como flexões, dependências, etc.

  • Tokenization: Processo de separar a sentença em suas partes: palavras, pontos, símbolos, etc.

  • Parts-of-Speech (POS) Tagging: Adiciona tags a cada token, como por exemplo, se é verbo, substantivo, adjetivo, etc.

  • Lemmatizing (Lemma): Traz a palavra na sua flexão, de modo que possam ser analisadas juntas.

  • Dependency Parsing: Encontra relação entre palavras “pais” e “filhos”.

2.5 Modelo

  • Análise:

    • Verbo? Substantivo? Quais são as flexões? Quais as dependências?

  • Um modelo é um banco de dados linguístico.

  • Específico de cada idioma.

  • Maioria das plataformas de PLN tem seus próprios modelo ou usam de terceiros.
